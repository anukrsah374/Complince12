AUTOMATED AGENT INSTALLER for INDEFEND AGENT - v2.3.2
----------------------------------------------------------------------------------------
Release Date - 1st March 2025
----------------------------------------------------------------------------------------
Works for 32 and 64 bit versions of windows 7, Windows 10, Windows 11
----------------------------------------------------------------------------------------
Works for inDefend Advanced Agent and SAM Agent
----------------------------------------------------------------------------------------

Note:
- In case of /silent and Manual Installation, buildVersion.property and RemConfig.ind files should be present 
- In case of /standalone : inDefend Installer zip and all the pre-requisite should be present
----------------------------------------------------------------------------------
Parameters:
-----------

automated_agent_installer.exe can be run with manual click

OR with CMD command below:

automated_agent_installer.exe /silent
automated_agent_installer.exe /standalone
automated_agent_installer.exe /silent /uninstalloff
automated_agent_installer.exe /standalone /unistalloff



Note : Require atleast 4GB RAM on system to run this tool
